To connect to Solana, we can pass phantom.import { DynamicContextProvider, DynamicWidget, SortWallets,defaultNumberOfWalletsToShow } from '@dynamic-labs/sdk-react';.... <DynamicContextProvider  settings={{ ... walletsFilter: SortWallets(['phantomevm', 'metamask', 'walletconnect', 'coinbase']); defaultNumberOfWalletsToShow: 4; ... } }}> Step 5: Set Phantom as the recommended wallet for new to web3 usersIn addition to setting Phantom as the top wallet, you can also configure it to be the recommended wallet for users that click the Get your first wallet button.
