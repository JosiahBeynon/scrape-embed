The following is a list of all possible error codes and their meanings. These error messages are inspired by Ethereum's EIP-1474 and EIP-1193.CodeTitleDescription4900DisconnectedPhantom could not connect to the network.4100UnauthorizedThe requested method and/or account has not been authorized by the user.4001User Rejected RequestThe user rejected the request through Phantom.-32000Invalid InputMissing or invalid parameters.-32002Requested resource not availableThis error occurs when a dapp attempts to submit a new transaction while Phantom's approval dialog is already open for a previous transaction. Only one approve window can be open at a time. Users should  approve or reject their transaction before initiating a new transaction.-32003Transaction RejectedPhantom does not recognize a valid transaction.-32601Method Not FoundPhantom does not recognize the method.-32603Internal ErrorSomething went wrong within Phantom.Typically, these errors will be easily parseable and have both a code and an explanation.
