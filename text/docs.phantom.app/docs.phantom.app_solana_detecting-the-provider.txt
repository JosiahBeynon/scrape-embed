Phantom's browser extension and mobile in-app browser will both inject a phantom object into the window of any web application the user visits, provided that site is using https://, on localhost, or is 127.0.0.1. Phantom will not inject the provider into iframes or sites use http://.If a phantom object exists, Solana apps can interact with Phantom via the API found at window.phantom.solana. This solana object is also available at window.solana to support legacy integrations.
